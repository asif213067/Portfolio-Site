# 💼 Responsive Personal Portfolio

A modern and fully responsive **Personal Portfolio Website** built with **HTML, CSS, and Bootstrap**. This website showcases my projects, technical skills, education, and contact information through a clean and user-friendly interface.

---

## 📌 Overview

This project is a personal portfolio landing page designed to represent my skills and work as a web developer. The website provides visitors with an easy way to explore my projects, learn about my skills and educational background, and contact me.

The entire website is fully responsive and optimized for desktop, tablet, and mobile devices.

---

## ✨ Features

* 🧭 Responsive navigation bar
* 🖼️ Modern hero banner with image and content
* 💼 Portfolio section showcasing projects
* 🛠️ Technical skills section
* 🎓 Education information section
* 📩 Contact form
* 📱 Fully responsive for all devices
* 🎨 Clean and modern user interface
* ⚡ Lightweight and fast-loading design

---

* Bootstrap
